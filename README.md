 # joycon-ppt
用JoyCon(R)作PPT翻页笔

## Screenshot
![Screenshot](https://raw.githubusercontent.com/hcreak/joycon-ppt/master/screenshot.jpg)

## 按键映射
| 按键   | 功能               |
| ------ |      ------       |
| ZR     | 回车（翻页）       |  
| R      | Alt+Tab(保持状态)  |  
| X Y A B  |  ↑ ← → ↓        | 
| +      | 放映幻灯片（F5）   |
| 摇杆   | 控制鼠标           |
| 摇杆按下 | 激光笔（保持）    |
| HOME   | 退出放映（Esc）    |     

## 说明
先通过蓝牙连接JoyCon(R)
Windows以识别出HID设备为准

当焦点处于窗口 按 `+` 键（ 即 `F5` ）可切换隐藏显示Log信息 用于Debug 功能开启后按 `HOME` 键（ 即 `Esc` ）可关闭
