## 按键映射
| 按键   | 功能               |
| ------ |      ------       |
| ZR     | 回车（翻页）       |  
| R      | Alt+Tab(保持状态)  |  
| X Y A B  |  ↑ ← → ↓        | 
| +      | 放映幻灯片（F5）   |
| 摇杆   | 控制鼠标           |
| 摇杆按下 | 激光笔（保持）    |
| HOME   | 退出放映（Esc）    |
